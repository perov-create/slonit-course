## Работа с Git

Команды с которыми я познакомился:

### Базовые команды git
1. **`git init`**  
   
2. **`git clone`**  
   
3. **`git add <файл>`**  
   
4. **`git commit -m "<сообщение>"`**  

5. **`git commit -a`**  
   
6. **`git status`**  

7. **`git log`**  

8. **`git branch`**  

9. **`git checkout <ветка>`**  

10. **`git pull`**  

11. **`git push`**  

### Флаги
- **a** - Автоматически индексирует все изменения
- **m** - Позволяет указать сообщение для коммита
